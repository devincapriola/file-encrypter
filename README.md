# python-encrypter
Python file encrypter using Fernet

<br/>

### Quick Start
clone the repository
```
git clone https://github.com/devincapriola/python-encrypter.git
```
install the requirements
```
pip install -r requirements.txt
```
run the encrypter
```
python encrypt.py
```
run the decrypter
```
python decrypt.py
```
